在windows上用putty连接

1 首先在Linux下安装SSH服务并配置

  1. 卸载重装一遍ssh服务

    sudo apt-get remove openssh-server
sudo apt-get install openssh-server

  2. 编辑sshd_config文件，修改几处配置才能正常使用用户名/密码的方式连接

    sudo vi /etc/ssh/sshd_config

    Port 22 #默认即可，如果有端口占用可以自己修改
    PasswordAuthentication yes # 允许用户名密码方式登录

  3. 修改完之后重启ssh服务

    sudo service ssh restart

    现在可以使用ssh客户端连接wsl了。

    2 然后在windows上安装Putty

    下载https://www.chiark.greenend.org.uk/~sgtatham/putty/latest.html
    正常安装即可，
    在windows 的cmd中输入putty回车，选则Session直接输入IP和端口就行了。可以保存配置。

    3 导入配置（配色方案等）

使用WSL的桌面

1. WSL中使用中科大镜像 更新 软件

修改源：sudo vim /etc/apt/sources.list

# 默认注释了源码镜像以提高 apt update 速度，如有需要可自行取消注释
deb https://mirrors.ustc.edu.cn/ubuntu/ bionic main restricted universe multiverse
# deb-src https://mirrors.ustc.edu.cn/ubuntu/ bionic main restricted universe multiverse
deb https://mirrors.ustc.edu.cn/ubuntu/ bionic-updates main restricted universe multiverse
# deb-src https://mirrors.ustc.edu.cn/ubuntu/ bionic-updates main restricted universe multiverse
deb https://mirrors.ustc.edu.cn/ubuntu/ bionic-backports main restricted universe multiverse
# deb-src https://mirrors.ustc.edu.cn/ubuntu/ bionic-backports main restricted universe multiverse
deb https://mirrors.ustc.edu.cn/ubuntu/ bionic-security main restricted universe multiverse
# deb-src https://mirrors.ustc.edu.cn/ubuntu/ bionic-security main restricted universe multiverse
deb https://mirrors.ustc.edu.cn/ubuntu/ bionic-proposed main restricted universe multiverse
# deb-src https://mirrors.ustc.edu.cn/ubuntu/ bionic-proposed main restricted universe multiverse

更新软件

sudo apt-get update 更新软件列表
sudo apt-get upgrade 更新软件

2. 在WSL上安装xfce desktop

这种桌面也可以，更接近ubuntu：https://blog.csdn.net/L141210113/article/details/103201152
最后wsl中运行 sudo compiz

1）安装 xfce desktop

sudo apt-get install xfce4-terminal
sudo apt-get install xfce4

2）配置文件：sudo vim ~/.bashrc，加入下面两行

export DISPLAY=:0.0
export LIBGL_ALWAYS_INDIRECT=1

保存退出后，source ~/.bashrc

3. 在windows上安装软件vcxsrv

软件下载链接：https://sourceforge.net/projects/vcxsrv/
运行.exe，选择one large window和 下面display number设置成0，

在WSL中运行sudo startxfce4，然后window下的vcxsrv就可以看到WSL的桌面了，
