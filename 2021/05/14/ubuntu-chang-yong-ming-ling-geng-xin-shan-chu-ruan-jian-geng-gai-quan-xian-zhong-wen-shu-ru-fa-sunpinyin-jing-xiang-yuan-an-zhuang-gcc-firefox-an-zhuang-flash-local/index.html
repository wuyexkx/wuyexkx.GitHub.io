1.获取root权限，修改只读文件

1.  sudo su   ---输入密码
2. cd /etc/default  切换到指定路径
3. sudo gedit xxx.conf

2.修改root密码

sudo passwd root

3.获取本机IP

设置环境变量： **gedit ~/.bashrc**
ERROR： Couldn't find an AF_INET address for [xxx]
**hostname -I**   获取IP
在bashrc文件后面加上下面的内容：
**export ROS_IP=`hostname -I`**   '用IP替换'
可以在环境变量~/.bashrc文件末尾加入**export ROS_IP=IP** ，避免每次打开terminal都要重复上面两句。

4.环境变量

**env** 或者 **export** 查看所有环境变量
**echo $PATH** 查看单个环境变量
**export VAR_NAME=value** 命名一个新的环境变量

4.1 环境变量作用域

1.只在当前终端，其他终端打开无效

在终端输入：
**export XXX_PATH=value**

2.当前用户有效，其他用户无效

**gedit ~/.bashrc**  然后在打开的最后追加 **export XXX_PATH=value**
添加完成保存，不能立即生效，需要重启或者**source ~/.bashrc**

3.所有用户有效

**sudo gedit /etc/profile**  然后追加**export XXX_PATH=value**
添加完成保存，不能立即生效，需要重启或者**source /etc/profile**

更改权限

sudo chmod -（代表类型）×××（所有者）×××（组用户）×××（其他用户）

-ld表示修改文件夹权限

更改文件的权限

sudo chmod 644 ××× （所有者有读和写的权限，组用户只有读的权限）
sudo chmod 777 ××× （每个人都有读和写以及执行的权限）
sudo chmod 444 ××× （每个人都只有只读权限）

0 [000] 无任何权限
4 [100] 只读权限
6 [110] 读写权限
7 [111] 读写执行权限

中文输入法SunPinYin

https://www.cnblogs.com/zhaopengcheng/p/6040605.html

  1. 安装语言包
  2. ibus输入法安装 
    安装ibus输入法
sudo apt-get install ibus ibus-clutter ibus-gtk ibus-gtk3 ibus-qt4

来安装ibus框架。用
im-config -s ibus

切换到ibus框架。再安装拼音引擎：
sudo apt-get install ibus-pinyin

再调出ibus Preference来添加该拼音输入法，一次不行再输入一次试试：
sudo ibus-setup    弹出对话框
  3. 系统输入法设置
    在System Settings–Text Entry中添加上ibus拼音输入法，并将Show current input source in the menu bar勾选上，这样就会在系统的菜单条上显示输入法切换的图标了。反复折腾就对了。

镜像源

版本一致

首先注意镜像源的内容必须跟ubuntu的版本对应，代号不一致会导致安装依赖库版本不一样，各种乱七八糟问题：

$ lsb_release -a  # 查看自己的ubuntu版本
	No LSB modules are available.
	Distributor ID:	Ubuntu
	Description:	Ubuntu 20.04.1 LTS
	Release:	20.04
	Codename:	focal

Ubuntu 16.04代号为：xenial
Ubuntu 17.04代号为：zesty
Ubuntu 18.04代号为：bionic
Ubuntu 19.04代号为：disco
Ubuntu 20.04代号为：focal
